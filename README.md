# AI Search Engine
This project is an AI-powered semantic search engine that retrieves results based on meaning rather than exact keywords.

## Features
* Semantic search using AI (not keyword-based)
* Fast and relevant results
* Simple and interactive UI using Streamlit

## Tech Stack
* Python
* Streamlit
* txtai
* Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run main_streamlit.py

## Project Structure
* main_streamlit.py → Main application
* dataset_text.py → Data handling
* seth-data.csv → Dataset
